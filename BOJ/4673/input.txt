## 입력은 없다


문제이해

자리수에 따라  d(n)  을 구할 수 있다
d(n)을 구하고 배열로 저장한 우에 (임의의 배열에 d(n)값을 push한다)
i=1~10000 까지 d(n)에 없다면 출력하면 된다
(즉, i=42 이고 d(n)값이 저장되어 있는 배열에 42가 없으므로 출력 = 매직넘버)

## 배열에 해당 인자가 없다_ 어떻게 구할까?
1) if ( !arr.includes(i) )
2) if ( arr.indexOf(i) === -1 )

## d(n) 의 계산
n이 한자리수면? d(n) = n+n = 2n

n이 두자리수면? d(n) = n+십+일

n이 세자리수면? d(n) = n+백+십+일

n이 네자리수면? d(n) = n+천+백+십+일

n이 다섯자리수(=10000)면? d(n) = 10000+1 = 10001
